# **App Name**: Muawiya Quranic Excellence

## Core Features:

- Access Approval Hub: A centralized gateway where the supervisor approves or rejects registration requests for students and teachers to ensure a secure and moderated learning environment.
- Intelligent Progress Tool: Uses AI as a tool to analyze 'Memorization' and 'Revision' logs, generating insights and personalized performance summaries for teachers and admins to identify students needing support.
- Dynamic Halaqa Orchestrator: Enables administrators to create educational rings and assign teachers; restricted views ensure teachers only interact with students within their designated group.
- Bimodal Educational Logs: Strictly separated data entry interfaces for new memorization (Al-Hifz) and daily revision (Al-Muraja'a) to maintain accurate pedagogical history.
- Financial Insight Module: A focused dashboard for tracking income and expenses with simple categorization for quick auditing and performance overview.
- Teacher Legacy Archive: Digital documentation system for staff performance history, salary records, and document archiving accessible to the supervisor.
- Watermarked Command Center: A mobile-responsive dashboard featuring the center's official watermark and consistent back-navigation controls for seamless usability.

## Style Guidelines:

- Primary color: Radiant Gold (#EAB308) evoking prestige and divine light, used for active states and critical interface markers.
- Background: Midnight Onyx (#0D0C0A), a deep charcoal with subtle golden undertones to maintain the requested dark theme with an elegant feel.
- Accent: Burnt Copper (#CC6228), an analogous hue that provides strong contrast against the dark background for call-to-action buttons.
- The user-requested 'Cairo' font is recommended for both headlines and body text to provide cultural relevance and modern readability. Note: currently only Google Fonts are supported.
- Minimalist grid layout with the center's logo integrated as a faint 5% opacity watermark centered behind the dashboard content.
- Monochrome gold-tinted icons from a consistent geometric set, designed to scale gracefully on mobile screens.
- Fluid 'fade-and-slide' transitions between management rings to reinforce a professional and polished app experience.
